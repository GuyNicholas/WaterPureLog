# WaterPureLog

A privacy-preserving decentralized application built on FHEVM (Fully Homomorphic Encryption Virtual Machine) for securely recording and tracking water quality data.

## Features

- Encrypted water quality data recording (TDS, pH, conductivity, hardness)
- Privacy-preserving on-chain storage
- Gamified badge system
- Rating history tracking
- Encrypted summary generation

## Project Structure

- `contracts/` - Solidity smart contracts
- `frontend/` - Next.js frontend application
